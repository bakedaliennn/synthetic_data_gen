# Synthetic data generation project
This project aims to generate synthetic marketing data so to recreate a functional marketing dashboard. This task is part of a recruitment process; the detailed initial instructions can be found [here](https://github.com/ej29-r3d/Marketing-Analytics-Assignments/tree/main/tcs-assignment).

---

Overall, the project consisted of:
1. Understanding marketing context and metrics
2. Describing the data pipeline requirements to maintain a marketing dashboard
3. Figuring out features to make the data realistic.
4. Generating synthetic data
5. Validating the data
6. Making an exploratory data analysis (EDA)
7. Building a Tableau dashboard
8. Make a 1-2 video presentation to showcase the dashboard and insights that can be extracted from it
