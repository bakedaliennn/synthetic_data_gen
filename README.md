# Marketing Dashboard Assignment
Recreate a functional marketing dashboard with Python-generated synthetic data.
This project is part of a recruitment process; the detailed initial instructions can be found [here](https://github.com/ej29-r3d/Marketing-Analytics-Assignments/tree/main/tcs-assignment).

---

Overall, the project consisted of:
1. Understanding marketing context and metrics
2. Describing the data pipeline requirements to maintain a marketing dashboard
3. Generating synthetic data
4. Validate the data
5. Make an exploratory data analysis (EDA)
6. Make a 1-2 video presentation to showcase the dashboard and insights that can be extracted from it
